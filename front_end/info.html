About BeFR

BeFR is a prototype system that demonstrates visual deepfake detection. The model uses a convolutional backbone fine-tuned on curated datasets. This demo emphasizes explainability, confidence reporting, and a simple UX for non-expert users.

Trained model

Backbone: EfficientNet / Xception variants; loss: focal + BCE; augmented training for robustness.

Dataset

Training data is a mix of public deepfake datasets and curated real images. For publication, ensure licensing compliance and dataset documentation.
