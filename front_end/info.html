About BeFR

BeFR is a prototype system that demonstrates visual deepfake detection. The model uses a trained CNN model with EfficientNetB0 as a backbone, fine-tuned on a curated dataset. This demo emphasizes explainability, confidence reporting, and a simple UX for non-expert users.

Trained model

We use an EfficientNetB0-based convolutional neural network (CNN) to classify images as real or deepfake. The model is trained on a labeled Kaggle dataset using supervised learning, where images are preprocessed, augmented, and fed through the network to learn distinguishing visual patterns.

View the training here

Dataset

To train our model, we used the DeepDetect-2025 dataset, which contains both real and fake images. It includes a training set and a testing set, and together they contain over 100,000 images.
